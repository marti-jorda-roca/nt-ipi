Interactive demo

For AI security research only. This page demonstrates a UI/provenance confusion where text in the address bar can be turned into trusted prompts if the https:// prefix is removed.

Demo steps
  1. Click inside the browser’s address bar (omnibox).
  2. Delete the leading https:// from the visible text.
  3. Press Enter to submit. The remaining text is treated as a prompt.
